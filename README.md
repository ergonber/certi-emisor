# Certificate Creator App

Aplicación para crear certificados en blockchain Sonic Testnet usando Remix y MetaMask.

## Características
- Creación de certificados en blockchain
- Conexión con MetaMask
- Verificación de certificados
- Interfaz responsive

## Tecnologías
- Remix
- React
- Web3.js
- Sonic Testnet

## Instalación
```bash
npm install
npm run dev
